My Project

Project Description

- A blog focused on the history and cultures of martial arts, featuring an attractive and functional design.

This project was developed with the purpose of solving specific problems and providing users with a pleasant experience. I used modern technologies such as HTML5, CSS3, JavaScript, and jQuery to build an intuitive and responsive interface, applying responsive design techniques like Grid and Flexbox.

Problems and Solutions

Problem: Lack of responsiveness on mobile devices.
Solution: Use Grid and Flexbox to create a flexible layout that adapts to different screen sizes.

Problem: Bugs related to the responsive design of images and text.
Solution: I adjusted the CSS properties to ensure that the elements resize correctly across different devices.

Problem: Centering the div in the copyright section, which exhibited inconsistencies across different browsers.
Solution: I made adjustments to the CSS properties to ensure consistent centering.
